Gestión de Losas Deportivas:

Sistema web destinado a la administración y reserva de losas deportivas, permitiendo un control centralizado, ordenado y accesible para usuarios y administradores.
Fue desarrollado en Java (NetBeans) con conexión a una base de datos en MySQL (phpMyAdmin).

Objetivo del Sistema:

Resolver los problemas comunes en la gestión manual de reservas, tales como:

Duplicidad de horarios y conflictos con clientes.
Falta de organización y registro histórico.
Baja accesibilidad para consultar disponibilidad.
Procesos lentos que requieren atención presencial o manual.
El sistema permite administrar reservas digitales de forma rápida, segura y automatizada.

Funcionalidades Principales

- Registro y administración de reservas.
- Consulta de disponibilidad de losas en tiempo real.
- Historial de uso y control de usuarios.
- Automatización del proceso de confirmación.
- Evita duplicidad de horarios.
- Gestión para clientes y administradores.
